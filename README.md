# 🧮Simple Calculator (CLI)

A command-line calculator built in Python that performs basic arithmetic operations with clean error handling and a user-friendly interface.

---

## 📌Description

This project is a simple yet functional calculator that runs entirely in the terminal. It takes two numbers and an operator as input and returns the result instantly. Built as part of a Python internship project to practice basic syntax, user input/output and conditional logic.

---

## Features

- Addition, Subtraction, Multiplication, Division
- Division by zero error handling
- Invalid input handling
- Clean CLI interface
- Type `exit` to quit anytime

---

## Tech Stack

| Tool | Purpose |
|------|---------|
| Python | Core language |
| if-else | Operation logic |

---

## How to Run

```bash
git clone https://github.com/gunn0067/simple-calculator.git
cd simple-calculator
python calculator.py
```
